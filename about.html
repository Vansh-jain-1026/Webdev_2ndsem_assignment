About the Project

NoteIt is a simple full-stack notes application. You can create, read, update, and delete notes. All data is stored in a real PostgreSQL database through Supabase. The backend is built with Python and Flask, and the frontend is plain HTML, CSS, and JavaScript.

Built as Capstone Project — Semester 2, Rishihood University.
